\# HelpGrailed Bot



Telegram бот для продажи аккаунтов и услуг.



\## Установка



1\. Клонировать репозиторий

2\. Создать `.env` файл с токенами

3\. Установить зависимости: `pip install -r requirements.txt`

4\. Запустить: `python main.py`



\## Переменные окружения



\- `BOT\_TOKEN` - токен бота от @BotFather

\- `CRYPTO\_TOKEN` - токен CryptoPay

\- `ADMIN\_IDS` - список ID администраторов
